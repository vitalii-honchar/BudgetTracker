# Budget Tracker iOS

A native iOS application for tracking personal expenses, built entirely with **[Claude Code](https://claude.ai/code)**.

## Features

- Track expenses with customizable categories
- Multi-currency support
- Date-based spending reports with visual breakdowns
- On-device AI-powered spending insights
- 100% private - all data stays on your device

## Technologies

- SwiftUI
- Core Data
- iOS 17+

## Getting Started

### Requirements

- Xcode 15+
- iOS 17+ device or simulator
- macOS with Apple Silicon or Intel

### Building

```bash
# Build the project
xcodebuild -scheme BudgetTracker -project BudgetTracker.xcodeproj build

# Or open in Xcode
open BudgetTracker.xcodeproj
```

See [CLAUDE.md](CLAUDE.md) for detailed development instructions.
